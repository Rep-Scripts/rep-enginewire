
# REP-ENGINEWIRE

## Installation
```bash
    git clone https://github.com/Rep-Scripts/rep-enginewire.git
    cd rep-enginewire/web
    pnpm i
    pnpm build
```

## Example
```lua
    local success = exports["rep-enginewire"]:Minigame()
```
